pub fn rfind_str(text: &str, pattern: &str) -> Option<usize>
Expand description

rfind_str finds the last ocurrence of pattern in the text and returns the index of the start of the match.

As of this writing, this function uses the two way algorithm in pure rust (with no SSE4.2 support).
