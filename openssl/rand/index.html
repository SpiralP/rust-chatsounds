Module openssl::rand

source ·
Expand description

Utilities for secure random number generation.

§Examples

To generate a buffer with cryptographically strong bytes:

use openssl::rand::rand_bytes;

let mut buf = [0; 256];
rand_bytes(&mut buf).unwrap();
